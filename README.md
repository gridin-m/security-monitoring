# Security Monitor для анализа ботнет трафика

## Описание проекта
Инструмент для автоматизированного мониторинга и анализа сетевых угроз на основе датасета CTU-13 (Neris botnet). Программа загружает реальные логи ботнет активности, анализирует подозрительные домены и проверяет их через VirusTotal API.

## Функциональность
- Загрузка и парсинг JSON-логов из датасета CTU-13
- Извлечение структуры доменов из ботнет трафика
- Выявление подозрительных доменов по характерным признакам
- Проверка доменов через VirusTotal API
- Имитация реагирования на угрозы (блокировка доменов, DNS-мониторинг)
- Формирование отчета в JSON и визуализация результатов

Программа создаёт
stratosphere_report.json — детальный отчет с информацией об угрозах
stratosphere_analysis.png — графики с топ-10 доменов и типами угроз
virus_total_cache.json — кэш для экономии лимитов API

Примечания
Бесплатный VirusTotal API позволяет делать 4 запроса в минуту
Программа автоматически кэширует результаты для экономии лимитов
Для работы необходим API ключ VirusTotal
Датасет CTU-13 содержит реальный ботнет трафик и используется в научных исследованиях
